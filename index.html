Frequently Asked Questions

What makes Flowcore different from traditional databases?

+

Flowcore uses event-first architecture where everything is written to the event store first, creating a single source of truth. Unlike traditional databases, we offer infinite event replay capabilities, allowing you to debug issues, test features, and train AI models with historical data at any point in time.

How does the infinite replay feature work?

+

Every event in Flowcore is preserved forever with immutable timestamps. You can replay events from any point in time to any destination, rebuild application state, test new features against historical data, or train AI models. This is perfect for debugging, auditing, and development iteration.

What databases and systems does Flowcore support?

+

Flowcore supports all major database types including relational, graph, time series, vector, key-value, column-based, and document databases. We also integrate with data warehouses, object storage, CSV/Parquet files, and communication systems like email, SMS, webhooks, and message queues.

How quickly can I get started with Flowcore?

+

You can get your first data core running in under 5 minutes. Simply install our CLI with npm install -g @flowcore/cli, create a data core with flowcore datacore create my-app, and start sending events. Our developer-friendly APIs and comprehensive documentation make integration seamless.

Is Flowcore suitable for production workloads?

+

Absolutely! Flowcore processes over 10 million events per second with sub-millisecond latency and 99.9% uptime. We offer enterprise-grade security with GDPR compliance, data obfuscation, permission management, and audit trails. Our auto-scaling infrastructure handles traffic spikes seamlessly.

What are the pricing options and how do they scale?

+

We offer flexible pricing starting at $400/month for the Standard plan with 100GB storage and 1,000 events/second. Our Medium and Large plans provide more storage and processing power, while our Cluster plan runs on your own servers. Extra storage is $0.10/GB/month and extra transformers are $20/unit/month.
